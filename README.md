# Statistics Concepts in Python

## 📌 Overview
This repository contains Python implementations of important statistical concepts and hypothesis testing techniques using Jupyter Notebook.

It is created for learning and practicing statistics concepts in Python.

---

## 📚 Topics Covered

### Statistics Concepts
- Mean
- Median
- Mode
- Variance
- Standard Deviation
- Skewness
- Kurtosis

### Hypothesis Testing
- Null Hypothesis
- Alternate Hypothesis
- P-Value
- Level of Significance
- Type I Error
- Type II Error
- Parametric Testing
